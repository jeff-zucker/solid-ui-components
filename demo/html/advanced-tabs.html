Alternate Ontologies

We can create a generic tabset like this:

:Tabset :hasTabs ( :Tab1 :Tab2 ) .

But sometimes we want to display pre-existing data in tabs, for example create one tab for each member of a team. Suppose our team data looks like this: 

:Team :member ( :Member1 :Member2 ) .
