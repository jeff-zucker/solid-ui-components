Creating an RDF tabset definition document

a. Define ontology prefixes

@prefix : <#> .
@prefix rdfs: <http://www.w3.org/2000/01/rdf-schema#> .
@prefix schema: <http://schema.org/> .
# add zero or more other ontologies

b. Define a tabset

:Tabset :hasTabs ( :Tab1 :Tab2 ) .
# add more tabs as desired, note : no commas between tabs.

c. Define tabs

:Tab1
  rdfs:label "Home" ;
  schema:url <https://example.org/home> .
  # add more properties as desired
# add more tabs as needed (remember to add them to the tabset too.)
