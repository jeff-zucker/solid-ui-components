Overview of the Solid-UI Tabs Component

Define a set of tabs in RDF or point to local or remote RDF data you want to display as a set of tabs. Then inject the tabs into a web page with an HTML widget.

To see how simple the process can be, look in the RDF and HTML tabs - they contain the complete code used to display the tabset you are viewing now.

To learn how to create & style horizontal and vertical tabsets, see the Create RDF and Create HTML tabs.

The Advanced Usage tab has instructions on creating custom actions on tab opening or closing and on using alternate ontologies and data structures.
