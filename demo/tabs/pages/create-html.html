Adding components to an HTML document

a. Create a tabs component

<div class = "solid-ui-tabs"          // required class
     data-source = "tabs.ttl"         // required RDF source
     data-backgroundcolor = "red"     // optional tab color
     data-selectedTab = 3             // optional selected tab number
     data-orientation = "left"        // optional placement of tabs
     data-subject = <url>             // optional see README
     data-predicate = <url>           // optional see README
     data-ordered = false             // optional see README
></div>

b. Import the solid-ui libraries

<script src="../solid-ui/lib/webpack-bundle.js"></script>
<script src="./solid-ui-components.js"></script>

// NOTE : libraries must be included after all components
